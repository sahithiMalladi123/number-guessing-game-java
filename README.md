🎯 Number Guessing Game

A simple and fun Java console-based Number Guessing Game where the player tries to guess a randomly generated number within limited attempts.

 FEATURES:
1.Random number generation
2.User input using Scanner
3.Hints like:
  Too High 📈
  Too Low 📉
4.Attempt counter
5.Winning message 🎉
6.Option to play again 🔄
7.Input validation using exception handling

 CONCEPTS USED:
1.Scanner Class
2.Random Class
3.Loops
4.Conditional Statements
5.Exception Handling
6.User Input Handling
7.Method Calling

📌 HOW THE GAME WORKS:
The program generates a random number.
The user enters a guess.
The program checks whether the guess is:
Correct
Too high
Too low
The game continues until:
The user guesses correctly, or
The attempts are completed.
The player can choose to play again.
